## Visão Geral

Sistema de **Retrieval-Augmented Generation (RAG)** especializado em engenharia elétrica, com capacidade de operação **totalmente offline**. Permite consultar normas técnicas, manuais e documentacao via chat com linguagem natural, sempre citando as fontes originais. Com **43 módulos Python**, o sistema abrange indexacao, busca semantica, geração de respostas e análise automatizada de documentos técnicos.

## Arquitetura RAG

### Indexacao e Embeddings
- Processamento de documentos em múltiplos formatos: **PDF, DOC, DOCX, TXT**.
- **OCR com Tesseract** para documentos escaneados ou com imagens de texto.
- Geração de embeddings com **Sentence-Transformers** (modelos leves para edge).
- Chunking inteligente com overlap para manter contexto entre fragmentos.

### Busca Semantica — FAISS
- **FAISS** (Facebook AI Similarity Search) como vector store de alta performance.
- Busca por similaridade cosseno em milhares de chunks em milissegundos.
- Re-ranking de resultados por relevancia e contexto da pergunta.

### Geração de Respostas — Ollama
- LLM local via **Ollama** com modelo **Mistral 7B** para operação offline.
- Prompts especializados com **20+ contextos de engenharia elétrica**.
- Respostas sempre acompanhadas de **citacoes das fontes** (documento, pagina, trecho).
- Opcao de integração com modelos online (OpenAI, Anthropic) para maior capacidade.

## Interface e API

### API REST — FastAPI
- Backend **FastAPI** com endpoints para: upload de documentos, consulta, análise e gerenciamento de índices.
- Documentacao automática OpenAPI/Swagger.
- Processamento assincrono de uploads e indexacao.

### Interface Web — Streamlit
- **10 paginas** na interface Streamlit para diferentes funcionalidades:
  - Chat com documentos (consulta por linguagem natural).
  - Upload e gerenciamento de documentos.
  - Análise de relatórios técnicos.
  - Análise de projetos elétricos.
  - Análise de qualidade de codigo.
  - Configurações e status do sistema.

## Funcionalidades de Análise

### Análise de Relatórios
- Identificacao automática de **riscos** e nao-conformidades.
- Verificação de **requisitos normativos** (NBR, IEC, ANSI).
- Destaque de pontos críticos e recomendacoes.

### Análise de Projetos
- Avaliação de conformidade com normas técnicas.
- Checklist automático de requisitos obrigatorios.
- Sugestoes de melhoria baseadas na base de conhecimento.

### Contextos Especializados (20+)
- Instalações elétricas de baixa e media tensao.
- Sistemas de proteção e coordenação.
- Qualidade de energia e harmonicos.
- SPDA e aterramento.
- Subestações e linhas de transmissão.
- Eficiencia energetica e sustentabilidade.

## Deploy

- **Docker Compose** com servicos isolados: API, Streamlit, Ollama.
- Funciona em maquinas sem acesso a internet (ideal para ambientes industriais).
- Configuração via variaveis de ambiente para adaptação a diferentes cenários.

## Diferenciais

- **Offline-first**: Opera sem internet, essencial para ambientes industriais e de campo.
- **Citacao de fontes**: Toda resposta referência o documento e trecho original.
- **Especializado**: Prompts e contextos otimizados para engenharia elétrica, nao genérico.
- **Modular**: 43 módulos com responsabilidades bem definidas, fácil de estender.
