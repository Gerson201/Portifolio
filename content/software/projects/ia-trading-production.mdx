## Visao Geral

Sistema de trading algoritmico de nivel profissional, projetado para operacoes intraday automatizadas no mercado financeiro. Construido com **Clean Architecture em 4 camadas**, o projeto abrange desde a ingestao de dados de mercado em tempo real ate a execucao de ordens com gestao de risco dinamica.

Com mais de **86 modulos** e cobertura de testes entre **93% e 97%**, o sistema demonstra maturidade de engenharia equivalente a ambientes de producao institucionais.

## Arquitetura

O sistema segue **Clean Architecture** com separacao rigorosa em 4 camadas:

- **Domain**: Entidades de mercado (Candle, Order, Position, Portfolio), regras de negocio puras e interfaces de repositorio.
- **Application**: Casos de uso orquestrados (analise de mercado, execucao de ordens, rebalanceamento), servicos de aplicacao e DTOs.
- **Infrastructure**: Integracao com MetaTrader 5, persistencia de dados, adaptadores de mercado e provedores de dados externos.
- **Presentation**: API REST via Flask, dashboard em tempo real com WebSocket e endpoints de monitoramento.

## Modelos de Deep Learning

### Predicao de Precos — LSTM com Attention
- Arquitetura LSTM multi-camada com mecanismo de **Self-Attention**.
- Treinamento com janelas deslizantes e features de microestrutura.
- Cobertura de testes do modulo: **95,5%**.

### Predicao de Volatilidade — GARCH + LSTM Hibrido
- Combinacao de modelo estatistico GARCH com rede LSTM para capturar tanto padroes lineares quanto nao-lineares.
- Utilizado para ajuste dinamico de stop-loss e dimensionamento de posicao.
- Cobertura de testes: **97,56%**.

### Agente de Reinforcement Learning — PPO
- Agente treinado com **Proximal Policy Optimization** por **1 milhao de timesteps**.
- Ambiente customizado compativel com Gymnasium, simulando o mercado com spread, slippage e custos operacionais.
- Reward shaping com penalizacao de drawdown e bonus por Sharpe Ratio.

## Gestao de Risco

- **Position Sizing**: Calculo baseado em ATR (Average True Range) combinado com **Criterio de Kelly** para alocacao otima.
- **Stop-Loss Dinamico**: Ajustado em tempo real pela predicao de volatilidade GARCH+LSTM.
- **Circuit Breakers**: Interrupcao automatica ao atingir limites de perda diaria ou drawdown maximo.
- **Monitoramento de Drawdown**: Tracking continuo com alertas em cascata.

## Qualidade de Codigo

- Mais de **100 testes unitarios** cobrindo todos os modulos criticos.
- Cobertura geral entre **93% e 97%** por modulo.
- Testes de integracao para o pipeline completo de dados ate execucao.
- Separacao total entre logica de negocio e infraestrutura, permitindo testes sem dependencias externas.

## Diferenciais Tecnicos

- Integracao nativa com **MetaTrader 5** para dados de mercado e execucao de ordens.
- Dashboard WebSocket para acompanhamento em tempo real de posicoes, P&L e sinais.
- Pipeline de dados com normalizacao, feature engineering e validacao automatica.
- Configuracao centralizada via YAML com parametros de modelo, risco e execucao.
