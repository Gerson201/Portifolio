## Visão Geral

O **ScrapBid** nasceu como uma ferramenta de scraping de precos para licitações públicas e evoluiu para uma **plataforma integrada de gestao empresarial** voltada a empresas que atuam em vendas B2B/B2G, projetos de engenharia e licitações públicas.

A aplicação e um **app desktop** (Electron + React) com backend local (FastAPI + SQLite), eliminando dependencia de cloud para operações senssiveis e garantindo performance mesmo offline.

## Arquitetura

### Frontend — Electron + React + TypeScript
- App desktop multiplataforma com **Electron**, empacotado como instalador nativo.
- Interface moderna com **React + TypeScript** e build via **Vite**.
- Comunicação com o backend via API REST local (localhost).
- UX focada em fluxos operacionais reais de empresas de engenharia.

### Backend — FastAPI + Python
- API REST com **FastAPI** para lógica de negocio, scraping e integração.
- **SQLite** como banco de dados local, com migrations gerenciadas pelo **Alembic**.
- Scrapers para o **Portal Nacional de Contratacoes Públicas (PNCP)** — coleta automática de editais e precos de referência.
- Módulo de scraping para **sistemas de fornecedores** via **Playwright** (automação headless).
- **OCR integrado** com pytesseract + pdf2image para extracao de dados de documentos PDF.

### Integração ERP
- Sincronização bidirecional com **Bling ERP** via API oficial.
- Importação/exportação de produtos, pedidos e clientes.
- Conciliacao de estoque e preco entre a plataforma e o ERP.

## Funcionalidades Principais

### Inteligencia de Licitações
- Coleta automática de editais do PNCP com filtros por categoria e regiao.
- Análise de precos de referência do governo para calcular margem de competitividade.
- Histórico de participacoes e resultados por cliente.
- **Calculadora de lances** com simulação de margem e imposto.

### Gestao Comercial B2B
- CRM simplificado para gestao de clientes e oportunidades.
- Pipeline de vendas com acompanhamento de propostas.
- Gerador de orcamentos com PDF automático.
- Integração fiscal para cálculo de impostos em projetos de engenharia.

### Dados e Relatórios
- Dashboard de desempenho comercial com gráficos de conversao.
- Relatórios de licitações ganhas/perdidas com análise de preco.
- Exportação para Excel/PDF para prestacao de contas.

## Diferenciais

- **Desktop-first**: Operação offline, sem latencia de rede, dados locais seguros.
- **Específico para engenharia**: Fluxos adaptados para empresas de projetos e servicos.
- **Integração PNCP**: Acesso direto ao banco de licitações públicas sem intermediarios.
- **Versionamento rigoroso**: Changelog completo desde v0.1.0 com breaking changes documentados.

## Contexto

Desenvolvido para atender necessidades reais de empresas de engenharia que participam de licitações públicas e precisam de inteligencia de precos para elaborar propostas competitivas.

## Decisões Técnicas

- **Electron** escolhido para garantir acesso a sistema de arquivos local e operação offline.
- **SQLite + Alembic** para banco local com migrations controladas — sem dependencia de servidor de banco de dados.
- **FastAPI** para backend de alta performance com tipagem forte via Pydantic.
- Separação clara entre frontend (porta Electron) e backend (FastAPI em processo separado) para facilitar testes e evolucao independente.
